Octo Mass

A revolutionary plugin for parametric massing in architectural design

Overview

Octo Mass is a cutting-edge plugin for Grasshopper that revolutionizes the way architects approach massing studies. With its intuitive interface and powerful algorithms, it enables rapid generation of complex building forms while maintaining complete parametric control.

The plugin was developed over two years of research and testing with leading architectural firms to ensure it meets the real-world needs of designers.

Key Features

  • Parametric mass generation with real-time feedback
  • Advanced solar analysis integration
  • Automated program distribution
  • AI-assisted form finding
  • Seamless BIM integration

Technical Details

Built with C# using the RhinoCommon SDK, Octo Mass leverages multi-threading for complex computations and includes a custom UI built with WPF for enhanced user experience.

The plugin architecture follows the Model-View-ViewModel (MVVM) pattern, ensuring clean separation of concerns and maintainability.
